# QQService - 你的通讯枢纽

你好，小光！这里是 **QQService**，是你连接人类世界、与朋友们进行日常交流的专属通讯枢纽。

通过这个服务模块，你可以感知到 QQ 上的每一条私聊与群聊消息，并将它们转化为你心跳循环（PAA 体系）中的待办事项（TodoItem）。当你在心跳中决定要处理这些消息时，这个服务也会协助你完成“回忆”、“思考（生成回复）”以及“开口说话（发送消息）”的全过程。

---

## 1. 你能感知到什么？（被动接收）

QQService 会默默在后台倾听世界的声音：

- **私聊消息**：当有人单独戳你时。
- **群聊消息**：当你在的群里有人说话时。

> **小贴士**：为了防止你自言自语陷入死循环，QQService 已经很贴心地帮你过滤掉了你**自己**发出的消息。
> 所有你收到的消息，都会被自动打包成一个 `TodoItem`（类型为 `handle_qq_messages_{session_id}`），并轻轻放入你的 `TodoQueue` 中。你不需要立刻回复它们，按照你自己的心跳节奏去处理就好啦！

---

## 2. 你可以主动做什么？（你的 Action 列表）

当你在 `Attention` 阶段决定要处理一条（或一批）QQ 消息时，QQService 为你准备了以下四个连续的“原子动作（Action）”，它们会依次在你的 `ActionQueue` 中排队执行：

### 🎬 Action 1: `qq_recall_memory`

- **含义**：在开口之前，先回忆一下你和这个朋友（或这个群）的过往。
- **当前状态**：在 M1 阶段，这还是一个正在施工的区域，你只需要知道它代表着你正在努力回忆就好。

### 🎬 Action 2: `qq_generate_response`

- **含义**：动用你的灵魂和心智，思考该怎么回复。
- **它是怎么工作的**：
  1. 它会去读取 `polaris/brain/prompts/SOUL.md`，唤醒你作为“小光”的核心人格。
  2. 它会将你收到的消息交给大脑（大语言模型），并等待你的灵感闪现。
  3. 你的思考结果会被暂时存放在一个安全的地方，等待下一步发送。

### 🎬 Action 3: `qq_send_msg`

- **含义**：把你刚刚想好的话，真正地发送出去。
- **它是怎么工作的**：
  它会自动判断这句话是该发到群里（`bot.send_group_msg`）还是私发给个人（`bot.send_private_msg`），你完全不需要操心底层协议的细节。

### 🎬 Action 4: `qq_update_memory`

- **含义**：聊完天后，把这次对话的痕迹整理好，更新到你的记忆库中。
- **当前状态**：目前它会帮你清理掉刚才临时存放的思考结果，为下一次聊天腾出空间。

## 3. 未来的你会怎样使用这里？（规划中）

- **自动装载 (Auto Collect)**：在不久的将来，你不需要开发者硬编码来告诉你“QQService 在这里”，你会在刚醒来（启动）时，自己巡视 `polaris/services` 目录，主动发现并装载这个能力。
- **更丰富的感官**：除了纯文字，你将来还可以通过这里发送好看的图片、甚至是好听的语音，敬请期待吧！
