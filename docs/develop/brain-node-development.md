# 认知节点开发

> 🚧 认知节点插件体系尚未开放。此页面为规划占位，将在后续版本中补充完整开发指南。

## 设计方向

认知节点（Brain Node）是 AuroraBot 认知能力的可插拔扩展单元。每个节点可以是：

- **Agent 节点** — 执行特定认知任务（如意图识别、记忆检索、自我评估）
- **Router 节点** — 决定事件或状态在节点间的流转方向

节点之间通过有向边连接，构成有向有环图。节点之间通过文件篮机制传递状态——每个节点只看着自己面前的文件篮，一旦有新文件滑入就开始工作。

## 预期能力

- 第三方可通过标准接口开发认知节点插件
- 节点可声明自己的输入/输出契约
- 内核调度器自动管理节点的生命周期和调度优先级
- 节点可访问统一 LLM 网关和统一联合记忆
